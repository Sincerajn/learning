18.13

任意组件间通讯

实现任意组件之间的通讯，和子父通讯的方式有些类似，都是通过监听自定义事件的方式，不同之处在于任意组件都可以通过一个统一的中间层，也可以把它理解为全局的 Vue 对象。

18.12

组件，子父通讯

$emit 方法向父组件传递自定义事件，父组件执行自定义事件的同时能获取其传参；事件是子父组件通讯的桥梁，传参是二者通讯的数据。

18.11

组件，父子通讯

设置自定义属性，并通过 props: Array[string] 调用该属性即可获取值。将自定义属性和父元素数据绑定，自定义属性成为父子组件通讯的桥梁，其值为二者通讯的数据。

18.10

配置组件

Vue.component() 定义的组件，内部 data 应该为一个返回对象的函数。template 也可以传入元素选择器。

18.9

组件，全局与局部组件

components 内定义局部组件，组件的 template 相当于 el。Vue.component(name: string, Vue: object) 定义全局组件，可在任何已实例化 Vue 的元素内直接使用。
